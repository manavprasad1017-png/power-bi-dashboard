# Power BI Sales Dashboard

## Project Overview
This project is a Power BI dashboard created to analyze sales performance, profit, and customer trends using interactive visualizations.

---

## Objectives
- Analyze total sales and profit
- Identify top-performing categories
- Track monthly sales trends
- Compare regional performance

---

## Tools Used
- Power BI
- Excel
- DAX

---

## KPIs
- Total Sales
- Total Profit
  

---

## Charts & Visuals
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Top Products
- Customer Segment Analysis

---

## Slicers
- Region
- Category
- Year
- Segment

## Insights
- Technology category generated highest sales
- West region showed maximum profit
